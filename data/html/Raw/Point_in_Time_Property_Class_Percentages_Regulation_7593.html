Local Government Act

Property Class Percentages Regulation

 Prescribed percentages

1  For the purposes of the definition of "converted value of land and improvements" in section 5 of the Local Government Act the following percentages are prescribed:

(a) 10% in respect of property class 1;

(b) 35% in respect of property class 2;

(c) 40% in respect of property class 3;

(d) 34% in respect of property class 4;

(e) 34% in respect of property class 5;

(f) 24.5% in respect of property class 6;

(g) 30% in respect of property class 7;

(h) 10% in respect of property class 8;

(i) 10% in respect of property class 9.

1.see B.C. Reg. 93/93 in the B.C. Gazette Part II, March 30, 1993, p. 147

[Provisions of the Local Government Act, R.S.B.C. 1996, c. 323, relevant to the enactment of this regulation: section 6.6 (3) (b.1)]
