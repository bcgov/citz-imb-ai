Assessment Act
Port Land Valuation Regulation
B.C. Reg. 304/2010
actual value = previous actual value x (1 + inflation
adjustment)
where
previous actual value
=
the actual value of the designated port land as set out in the
assessment roll for the previous taxation year;
inflation adjustment
=
the percentage change in the consumer price index for July of
the year in which the assessment roll is completed from July
of the previous year.
