(2) For taxation years subsequent to the 2015 taxation year, the actual Value of designated port land for a taxation year is the actual Value determined by the following formula:
Actual value = previous actual value x (1 + inflation Adjustment)
Where
Previous actual value = The actual value of the designated port land as set out in the Assessment roll for the previous taxation year; Inflation adjustment = The percentage change in the consumer price index for july of The year in which the assessment roll is completed from july Of the previous year.
