B.c. reg. 285/98 O.c. 1095/98 Deposited august 4, 1998 Effective december 1, 1998 Fire services act British columbia fire code regulation Code adopted 1 The british columbia fire code which adopts with some changes the National fire code of canada 1995 as it existed on january 1, 1998, is adopted and enacted in the form attached to this Regulation.
British columbia fire code, 1998 Exempt from publication. this code may be purchased from Government publications, queen's printer, 1-800-663-6105 or (250) 387-6409, 2nd floor, 563 superior street, victoria, b.c. v8w 9v7.
Changes to the national code are published in the british columbia Gazette, part ii, volume 41, no. 16, page 569.
Note:
This regulation replaces b.c. regs. 492/59, 173/60, 175/62, 63/78, 15/87 and 403/92.
