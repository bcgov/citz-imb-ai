Deposited august 5, 2003 Effective september 1, 2003 Fire services act British columbia fire code regulation Code adopted 1 The national fire code of canada 1995, as it existed on january 1, 1998, is adopted as the british columbia fire code, with (a) the changes set out in the attachment to the british Columbia fire code regulation, made by b.c. reg. 285/98, except to The extent that the changes in that attachment are amended by Amendments set out in the schedule to this order, and (b) the amendments set out in the schedule to this order.
