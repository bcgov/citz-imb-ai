Code adopted
1
The National Fire Code of Canada 1995, as it existed on January 1,
1998, is adopted as the British Columbia Fire Code, with
(a) the changes set out in the attachment to the British
Columbia Fire Code Regulation, made by B.C. Reg. 285/98, except to
the extent that the changes in that attachment are amended by
amendments set out in the Schedule to this order, and
(b) the amendments set out in the Schedule to this order.
