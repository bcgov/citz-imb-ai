How data is stored in Neo4j

Graph Database and vector store

  • We us e a graph database to store the data and a vector store to store the embeddings.
  • Neo4J is used for both.
  • We use Neo4j to do more advanced queries that require clustering the data, finding communities which are much easier to implement using a graph database.
